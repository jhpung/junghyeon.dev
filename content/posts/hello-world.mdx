## 시작

안녕하세요! 이 블로그는 개발하면서 배운 것들을 기록하기 위해 만들었습니다.

## 기술 스택

이 블로그는 다음 기술로 만들어졌습니다:

- **Next.js 15** — App Router와 React Server Components
- **Tailwind CSS v4** — 유틸리티 퍼스트 스타일링
- **MDX** — 마크다운 + React 컴포넌트
- **Shiki** — 코드 구문 강조

### 프론트엔드

Next.js 15의 App Router를 사용하여 서버 컴포넌트와 클라이언트 컴포넌트를 적절히 분리했습니다.

#### 라우팅 구조

파일 시스템 기반 라우팅으로 페이지를 구성합니다.

#### 스타일링

Tailwind CSS v4의 CSS-based config를 사용하여 별도의 설정 파일 없이 관리합니다.

## 코드 예시

TypeScript로 간단한 함수를 작성해 보겠습니다:

```typescript title="greet.ts" {2}
function greet(name: string): string {
  return `안녕하세요, ${name}님!`;
}

console.log(greet("세계"));
```

Python도 지원합니다:

```python title="fibonacci.py" {2-3,7}
def fibonacci(n: int) -> list[int]:
    """피보나치 수열 생성"""
    if n <= 0:
        return []
    seq = [0, 1]
    while len(seq) < n:
        seq.append(seq[-1] + seq[-2])
    return seq[:n]

print(fibonacci(10))
```
